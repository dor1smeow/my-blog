# My Blog

一个基于 Next.js 16 构建的个人博客项目，主要用于记录前端开发、TypeScript、组件设计和内容型网站搭建相关的实践与思考。项目目前以静态内容展示为主，重点放在页面结构、组件复用和内容组织方式上。

## 项目介绍

这个项目是一个内容优先的博客站点，当前包含首页、文章列表页、文章详情页、关于页和 404 页面。文章数据统一维护在本地数据文件中，页面通过共享的数据层和 UI 组件完成渲染，整体结构比较轻，适合后续继续扩展内容管理、标签筛选和部署能力。

## 技术栈

- Next.js 16
- React 19
- TypeScript 5
- Tailwind CSS 4
- shadcn/ui
- Lucide React

## 已完成功能

- 首页展示博客定位、精选文章和常用主题标签
- 文章列表页统一展示全部文章卡片
- 动态文章详情页支持按 `slug` 渲染内容
- 文章页面已接入基础 SEO metadata
- 已实现静态参数生成与 404 兜底
- 已抽离全局布局、头部导航和页脚
- 已统一文章数据结构、标签提取、日期格式化和阅读时长计算逻辑
- 已提供关于页与联系方式展示

## 如何启动

1. 安装依赖

```bash
pnpm install
```

2. 启动开发环境

```bash
pnpm dev
```

3. 在浏览器访问

```text
http://localhost:3000
```

常用命令：

```bash
pnpm lint
pnpm build
pnpm start
```
